    Default¶

    Default PlantUML theme that contains light and dark flavors.

    Open Styles on GitHub

    How to use¶

    In order to use this theme with mkdocs_puml, set theme config of the plugin as follows:

    theme:
    light: default/light
    dark: default/dark
